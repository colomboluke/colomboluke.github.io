Projects

Connect 4 (Java)

Outside of class I recreated Connect 4, playable in console and on a GUI created with the Swing library. I used model, view, controlled architecture to allow for different boards, players. Currently adding capabilities to let user play against the computer, using the strategy design pattern.

RAFT Implementation (Python)

For my CS3700 final, we implemented the RAFT algorithm in a simulated distributed database. We had keep in mind query latency, consistency, and packet overhead, with simulations for various unreliable networks. Built using TCP/IP sockets in Python and object oriented programming.

FTP Client (Python)

A client that can connect to an FTP server and perform operations like copy, move, and make/remove directory. Runs on the command line using the argparse module.

Web Crawler (Python)

A web crawler/spider that traverses through the CS3700 class website. Intended to familiarize us with various HTTP methods as well as parsing HTML.

Snake (Racket)

My first personal project, made during my first ever CS class, was recreating Snake. I used a lot of the principles and design patterns we were learning (recursion, linked lists, abstraction) and had a lot of fun experimenting with coding on my own.

Reliable Transport Protocol (Python)

This project was to design a protocol on top of UDP sockets that would reliably transfer data packets between a sender and a receiver. This was heavily based on TCP, and was tested over various forms of unreliable networks.
